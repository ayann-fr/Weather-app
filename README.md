# Weather App 🌦️

A simple weather application built using HTML, CSS, and JavaScript.

## Features
- Search weather by city name
- Get weather using current location
- Displays:
  - City Name
  - Temperature
  - Local Time

## Technologies Used
- HTML5
- CSS3
- JavaScript
- WeatherAPI
